busca Sequencial -> é uma busca que acontece de forma Sequencial, ou seja, é realizada uma comparação entre o valor buscado e cada uma das posições do vetor em que se está buscando este valor. Ela pode ser ordenada, sendo este caso mais eficiente, uma vez que, estando o vetor ordenado, podemos pesquisar no vetor até o momento que o valor se torna menor que a posição atual, indicando que o valor buscado não se encontra naquele vetor, o que evita de percorrer todo o vetor para só no final para a busca.

Busca Binária -> Funciona apenas para vetores ordenados, e tem como primissa a frase "Dividir para conquistar". Dessa forma, este tipo de busca divide o vetor ao meio várias vezes, até que o valor procurado seja encontrado. Ele sempre realizada a comparação com o valor que está no meio do vetor. Caso seja igual, a posição é retornada, caso não, é feita uma verificação a fim de saber se o valor procurado é menor ou maior. Se maior, o vetor é "diminuido" para os usar apaenas os valores da esquerda. Da mesma forma, se maior, é "diminuido" para usar apenas os valores da direita. Isso é feito até que o valor seja encontrado ou o veotr tenha o tamanho de uma posição apenas. Caso não encontre retorna -1. É mais eficiente em alguns casos, visto que diminui a quantidade de comparações realizadas. Entretanto, por ser necessário que o vetor esteja ordenado, não se faz eficiente em situações em que o vetor é constantemente alterado com a introdução ou retirada de valores, o que necessitaria de uma ordenação a cada inerações, o que pode não ser interessante dependendo d asituação.
